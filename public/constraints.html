LDP.js

LDP.js is an LDP server built using Node.js and MongoDB. It accepts most any RDF, but does have a few constraints:

  • LDP.js does not yet support non-RDF source.
  • LDP.js does not support ldp:IndirectContainer.
  • Containers cannot be modified via PUT.
  • The interaction model of a resource cannot be changed by modifying its rdf:type after creation. If the resource was created as RDF Source, it cannot be changed to a container.
  • Deleting a container does not delete any of its resources.
DISCLAIMER: This service is provided as-is. It could go down at any time and change without warning. In fact, it might have already changed while you were reading this. The data behind this services will get scrubbed periodically. If this service gets abused, wse will either take it down or restrict access. Play nice.
